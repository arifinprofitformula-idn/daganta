# Roadmap Daganta Foundation v0.1

## Goal

Membangun fondasi teknis SaaS multi-tenant yang aman, ringan, dan siap bertumbuh.

## Build Order

1. Project skeleton
2. Environment setup
3. Prisma schema v0.1
4. Supabase connection
5. Seed tenant demo
6. Tenant resolver
7. Dashboard shell
8. Storefront shell
9. Security/RLS migration draft
10. Review and stabilize

## Not Included in v0.1

- Real payment gateway
- Real shipping API
- Real Dripsender integration
- Real Mailketing integration
- Custom domain automation
- Advanced billing automation
- Native mobile app
- AI content generator
- Marketplace sync
